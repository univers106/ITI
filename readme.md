# Contribute

## Ветки в репозитории
Разработка у нас ведется в ветке `main`, то есть от этой ветки отводим свои, делаем там изменения и пул реквестом вливаем в эту ветку (при пул реквесте все коммиты ветки сковаштся в один). Список веток:

* `main` - главная ветка (обязательные ПР с ревью);
* `users/<user_name>/<feature>` - ветки для фичей одного человека (проверяет флоу `branches.yaml`);
* `features/<feature>` - ветки для совместных фичей (проверяет флоу `branches.yaml`);
* `releases/<id>` - ветки для релизов (обязательные ПР с ревью);
